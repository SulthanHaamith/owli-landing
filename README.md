# Owl I Health and Wellness Center — Landing Page

A static landing page for a naturopathy clinic in Tirunelveli, Tamil Nadu, India.

## Quick Start

Open `index.html` in any web browser — no build step required.

## Deployment

Upload all files to Hostinger's `public_html/` directory via File Manager or FTP.

## Placeholder Content

Search for `[PLACEHOLDER]` comments in the HTML to find all items that need to be replaced with real content (photos, doctor details, testimonials, etc.). See `CLAUDE.md` Section 14 for the full checklist.
